# 📊 Data Jobs Project (Power BI)

## 🔹 Introduction
The **Data Jobs Project** is a complete end‑to‑end **data analytics case study** built in **Power BI**. It explores the **global data job market** using a dataset of job postings, salaries, skills, and locations.  

The project demonstrates how raw job data can be transformed into **interactive dashboards** that provide actionable insights for **job seekers, recruiters, and career transitioners**.

---

## 🔹 Project Objectives
- **Data Cleaning**: Prepare raw job postings for analysis.  
- **ETL Pipeline**: Extract, transform, and load data using Power Query.  
- **Data Modeling**: Build efficient relationships with a star schema.  
- **Dashboard Development**: Create multiple versions of dashboards to showcase iterative design.  
- **Insight Generation**: Highlight salary trends, skill demand, and job distribution.  

---

## 🔹 Versions
- **Dashboard 1**  
  - Multi‑page layout  
  - Exploratory design with detailed breakdowns  
  - Focus on job counts, salaries, and skill distribution
  - Job Drill through 

- **Dashboard 2 (Final Version)**  
  - Single‑page “mission control” design  
  - Cleaner visuals and recruiter‑friendly layout  
  - DAX, Measures and Parameters included

---

## 🔹 Key Insights
- **479K+ data jobs analyzed across 161 countries**  
- **Median yearly salary: $113K** vs **hourly: $48**  
- **Top skills in demand**: Python, SQL, AWS, Azure, Tableau  
- **Job type breakdown**: 78% Full‑time, 6% Contractor, rest Internship/Part‑time  
- **Global distribution**: Jobs mapped across North America, Europe, and Asia
